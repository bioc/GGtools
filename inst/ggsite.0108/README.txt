ggexpsite.0108 -- README.txt

1) htdocs dir holds html and png files used for the
presentation layer

2) cgi-bin holds R code for the cgi-bin layer

3) We assume you have CGIwithR installed and capable of
running everything in the trivial.* sources supplied with
CGIwithR.

4) You must also have Bioconductor GGtools and GGdata packages 
installed in the R used by CGIwithR

5) place the htdocs contents in your server's public
document folder, and the R code in the cgi executables folder

6) after installing the docs and CGI as described above, point
browser to localhost/ggexpl.html

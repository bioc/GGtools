GGexplorer ver 0.0, simple query facility.

The Bioconductor 2.0 packages GGdata and GGtools will be used to perform chromosome-wide screens for snps associated with expression of selected genes in 58 CEPH CEU founders.

Example queries that should resolve in a few minutes at most; choose to report on 25 SNPS:

  • chromosome 20, gene CPNE1 (cis)
  • chromosome 12, gene RPS26 (cis)
  • chromosome 13, gene TCEA1 (trans; gene on chr8)

NB: If the gene symbol that you submit is not found in the bioconductor hgfocusSYMBOL mapping, you will be asked to give another symbol. If the gene symbol is annotated to several probe sets, one is selected arbitrarily. In a future release we will allow direct specification of probe set identifier.
